# 🌦️ Weather Info Bot (AIML)

## Description
This is a simple AIML-based chatbot that provides static weather information for selected cities.

## Features
- Responds to weather queries
- Supports cities like Chennai, Bangalore, Delhi, Mumbai
- Simple rule-based chatbot

## Technologies Used
- Python
- AIML

## How to Run
1. Install AIML library:
   pip install python-aiml

2. Run the bot:
   python bot.py

## Sample Input
- hello
- weather in chennai
- weather in delhi
- bye
